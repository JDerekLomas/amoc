The Physics

Two-layer ocean model

Surface: dq/dt + J(ψ, q) = curl(τ) − rζ + A∇²ζ − αT∂T/∂x + F1(ψd−ψ)
Deep: dqd/dt + J(ψd, qd) = −rdζd + A∇²ζd + F2(ψ−ψd)
Ts: dT/dt = S·cosθz·αice − (A+BT) + κ∇²T − γ(Ts−Td)/H(x,y) + Qland
Td: dTd/dt = γ(Ts−Td)/Hd(x,y) + κd∇²Td

β western intensification · αice ice-albedo feedback · H(x,y) bathymetry
F1,2 interfacial coupling · Qland land-ocean heat exchange
