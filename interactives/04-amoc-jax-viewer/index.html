amoc-jax viewer — see the simulator

amoc-jax is a Python/JAX program; it doesn't run in the browser. So we ran it offline for several parameter scenarios and saved snapshot frames during the spin-up. Pick a scenario and scrub through it. Each frame is a real output of the simulator — what you'd see if you ran scripts/run.py locally with the same parameters.

—
ψ > 0 (anticyclonic, NH subtropical / SH subpolar) ψ < 0 (cyclonic)
0 / 7

What you're seeing

The colored field is the surface streamfunction ψs. Anticyclonic gyres (NH subtropical) are red; cyclonic (NH subpolar) are blue. Continents are masked off (dark). Each scenario starts from rest at frame 0 and evolves under that scenario's wind/β/viscosity. Steady state is reached around frames 5–8.

This is real simulator output, not a browser approximation. The physics ran in JAX — Arakawa Jacobian, FFT/DST Poisson solver, RK2 stepper, ERA5 wind curl, ETOPO mask. The interactives (Stommel, Sverdrup, Munk) implement simplifications of these equations in JS so they can run live in your browser. This page lets you see the full thing.

How to run it yourself

The Python source is in amoc-jax/ on GitHub. Reproducing any of the frames above:

cd amoc-jax
uv venv --python 3.12
uv pip install -e ".[dev]"
.venv/bin/python scripts/run.py --steps 12000 --save-every 1500 --out output/play

Frames rendered offline with scripts/render_scenarios.py. Re-run that script to refresh the frames after a parameter change.
